Guard.Me

Guard.Me is an application increasing User's safety. It provides bunch of tools, such as "Take Me Home" tool allowing User launch quick, one tap home navigation. Another usefull tool is called "Show Me Emergencies". It allows User to find emergency spots placed nearby. Another feature, optional "Guard Button", places special button on Screen Lock, containing informations about allergies, blood type, and contacts to trustworthy people, in case of emergency.
